ReadHistoryRequest

Only users can use this method. See code examples.

---functions---
messages.readHistory#0e306d3a peer:InputPeer max_id:int = messages.AffectedMessages

Returns

messages.AffectedMessages

This type can only be an instance of:

AffectedMessages

Parameters

peerInputPeerAnything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.).
max_idint

Known RPC errors

This request can cause 2 known errors:

PeerIdInvalidErrorAn invalid Peer was used. Make sure to pass the right peer type.
TimeoutErrorA timeout occurred while fetching data from the bot.

You can import these from telethon.errors.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.messages.ReadHistoryRequest(
        peer='username',
        max_id=0
    ))
    print(result.stringify())
